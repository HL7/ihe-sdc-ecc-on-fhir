Title:IHE SDC/eCC on FHIR Observation
Status:Active as of 2021-10-14T14:08:49-04:00
Definition:

A Observation Profile meant to capture quantity Question and Answer Sets from an IHE SDC Form. Any answer that has a quantity datatype in IHE SDC, or any datatype requiring a UCUM unit should be represented using this profile which requires the Observation.valueQuantity
